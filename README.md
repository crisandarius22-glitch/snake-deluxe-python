# Snake Deluxe – Python 🐍

Joc Snake realizat în Python folosind biblioteca Pygame, cu funcționalități extinse precum obstacole dinamice, sistem de scor și dificultate progresivă.

## 🎮 Funcționalități

* control în timp real prin tastatură
* detectarea coliziunilor (pereți, obstacole, auto-coliziune)
* generare aleatorie a hranei
* obstacole dinamice adăugate pe parcurs
* sistem de scor + salvare highscore în fișier
* creștere progresivă a dificultății
* efecte sonore (eat, game over, record nou)
* animație vizuală pentru scor (+1)

## 🛠 Tehnologii utilizate

* Python
* Pygame

## ▶️ Cum rulezi proiectul

1. Instalează Python (dacă nu ai deja)
2. Instalează biblioteca Pygame:

pip install pygame

3. Rulează jocul:

python snake.py
